## About Library Management

Project was developed for managing a small library

## Features
- Email based login
- Multiple users :- [Admin,Staff]
    - **Admin**
        - Manage Students
        - Manage Books
        - View report        
    - **Staff**
        - View Books
        - View Students
- Actions based on User type
- Issue and deposit Books
- Calculating return date
